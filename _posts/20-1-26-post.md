#### 1017_A.cpp
***
 ```cpp
  bool compare(const pair<int,int> &a, const pair<int,int> &b){
    if(a.first == b.first){
        return a.second<b.second;
    }
    return a.first > b.first;
}
```
compare 함수를 만들어 sort에 사용했다. first기준으로 오름차순, second 기준 내림차순.
실제로 알아만 뒀지 비교함수를 만들어 비교한적은 처음이다. 꽤 유용하므로 기억해두자.  
compare함수로 pair값을 비교하면 default로 first와 second로 정렬된다.
